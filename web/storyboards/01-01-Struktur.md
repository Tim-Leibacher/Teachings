# Storyboard HTML 1

| Punkt                    | Bildschirmdarstellung                          | Sprechertext                                                                                                            |
| ------------------------ | ---------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| 1. Einführung HTML       | Kurze Animation „HTML = Sprache für Webseiten“ | „HTML ist die Basis jeder Webseite. Jede Seite braucht ein Grundgerüst, damit der Browser sie richtig darstellen kann.“ |
| 2. `<!DOCTYPE html>`     | Zeige eine HTML-Datei, erste Zeile markiert    | „Ganz oben steht `<!DOCTYPE html>`. Das sagt dem Browser: Dies ist eine HTML5-Seite.“                                   |
| 3. `<html>` Tag          | Markiere die `<html>`-Tags um alles herum      | „Das `<html>`-Element ist das Wurzel-Element. Alles auf der Seite liegt innerhalb von `<html>` und `</html>`.“          |
| 4. `<head>` Bereich      | Markiere `<head>` inkl. `<title>`              | „Im `<head>` stehen Informationen über die Seite. Zum Beispiel der Titel, der im Browser-Tab angezeigt wird.“           |
| 5. `<title>` Tag         | Zeige Titel im Editor und Tab                  | „Der `<title>`-Tag bestimmt den Titel der Seite. Zum Beispiel 'Meine erste Webseite'.“                                  |
| 6. `<body>` Bereich      | Markiere `<body>` inkl. Beispieltext           | „Im `<body>` steht alles, was der Nutzer sieht: Texte, Bilder, Tabellen, Links.“                                        |
| 7. Kurze Zusammenfassung | Zeige komplettes Grundgerüst                   | „Zusammengefasst: `<!DOCTYPE html>`, `<html>`, `<head>` und `<body>` sind die Grundbausteine jeder HTML-Seite.“         |
