# Auftrag 2: Titel und Begrüssung – Inhalt sichtbar machen

## Ziel

Du ergänzt deine HTML-Seite mit einem sinnvollen Titel und einem sichtbaren Text im Body-Bereich.

## Beschreibung

Öffne deine bestehende `index.html` und füge im `<head>` einen passenden Titel hinzu, z. B. „Meine erste Webseite“.  
Schreibe im `<body>` einen kurzen Begrüssungstext wie zum Beispiel:  
„Hallo Welt, ich bin eine HTML-Seite.“  

Achte darauf, dass dein Text innerhalb der `<body>`-Tags steht.  
Probiere verschiedene Varianten aus und überprüfe jeweils im Browser, ob sich der Titel und der Text korrekt anzeigen.

## Erwartung

Eine korrekt strukturierte HTML-Seite mit einem sichtbaren Text und einem Titel, der im Browser-Tab angezeigt wird.

## Tipps

- Verwende die Vorschaufunktion im Browser, um Änderungen sofort zu sehen.  
- Achte darauf, dass du `<head>` und `<body>` nicht verwechselst.  
- Ergänze optional Überschriften (`<h1>`) oder Zeilenumbrüche (`<br>`), um dein Layout zu verbessern.  
- Wenn du in Visual Studio Code arbeitest, kannst du mit `Shift + Alt + F` den Code automatisch formatieren.
